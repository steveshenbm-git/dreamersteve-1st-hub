# Curation Card Template

Use this format to avoid asking the user to review every intermediate image.

```text
资料整理卡

任务：
来源范围：

建议确认：
1. [item] -> [recommended status]
   理由：
   未来用途：
   风险：

2. [item] -> [recommended status]
   理由：
   未来用途：
   风险：

自动归档，不建议人工处理：
- [item list]
  状态：process-only

需要用户确认：
- 是否确认 1 为 [status]？
- 是否确认 2 为 [status]？
- 哪些项目暂不入库？
```

Keep the card short. Prefer 3-6 confirmation items per task. If there are more, group them by reason.
